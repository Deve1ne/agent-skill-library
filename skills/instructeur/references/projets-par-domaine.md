# Projets GitHub et outils utiles par domaine

Liste curatée de projets de référence pour accélérer les tutoriels.
Consulter cette liste en Phase 2 pour trouver des projets existants.

> Cette liste est un point de départ. Toujours vérifier avec une recherche
> web que le projet est encore maintenu et qu'il n'y a pas de meilleure
> alternative récente.
